# 600036.SH（招商银行）— Investment Thesis

**Version:** v1 | **Date:** 2026-03-24 | **Status:** Active

> 中国零售银行的绝对龙头，以AUM财富管理护城河应对低利率时代的净息差压缩，当前6.8x PE / ~1.1x PB处于历史底部，5%+股息率提供下行保护；市场的"息差压缩叙事"低估了财富管理费收入的结构性增长，是本Thesis的核心预期差。

---

## Fundamental — 大方向

> 回答：基本面是否支持大方向？会持续多久？这是判断方向对不对的维度，不负责算 Risk/Return。

**当前状态：**
招商银行（招行）是中国股份制银行中资产规模最大（总资产13.07万亿元，+7.56% YoY）、零售业务最强的银行。2025年全年归母净利润约1,501.81亿元，同比微增，符合共识预期（1,500.08亿）。公司当前处于盈利能力防御阶段：LPR下调和存量房贷利率调整带来的净息差（NIM）收窄压力，正在被财富管理手续费的较快增长部分对冲。2025前三季度NIM 1.87%（同比-12bp），但财富管理手续费206.70亿元同比增长+18.76%——这两条线的剪刀差，是招行相对于同业的核心分化点。

**支撑大方向的主要因子：**

- **因子1：大财富管理护城河（核心正向驱动）**
  招行零售AUM超13万亿元，私行客户（资产1,000万+）和金葵花客户构成高黏性费用来源。利率中枢下行环境中，AUM规模积累带来的管理费收入具备结构性增长优势，是对NIM压缩的最重要对冲。2025H1财富管理费 +18.76% 明显领跑行业，验证了这一逻辑的持续性。在资本市场温和复苏的背景下（2026年A股/港股若延续修复），这一增速有望维持或加速。

- **因子2：资产质量领先（盈利稳定性的护城河）**
  招行不良率约0.9%、拨备覆盖率约400%，显著优于股份制银行平均水平。这意味着在信用周期下行时期，招行信用成本的波动性远低于同业，给盈利端提供缓冲。房地产和零售信贷敞口相对可控，2025年资产质量数据未出现恶化迹象。

- **因子3：净息差压力（主要负向因子，但预期已大部分定价）**
  LPR下调 + 存量房贷利率调整 + 信贷需求偏弱，是NIM收窄的三重压力来源。这是当前最大的不确定性，也是市场给予历史低估值的主要理由。管理层在2025 Q3业绩会上表示NIM已进入阶段性企稳区间，若无超预期的政策性降息（2026年再降LPR 30bp+），进一步大幅收窄的空间有限。

**Base Scenario（18个月最可能路径）：**
NIM在1.85-1.90%区间企稳，全年净利润小幅增长（+3-5%/年），财富管理手续费维持10-15%增速，资产质量保持稳定，全年分红率维持35%。随着2026年宏观经济温和复苏和资本市场活跃度回升，AUM管理费有望上修预期。目标价48 CNY（约8.3x 2026E PE / 1.35x PB），18个月总回报约+28%（含股息）。

**Adverse Scenario（最可能的不利情景）：**
LPR超预期再降50bp以上，且消费信贷/信用卡不良率明显上升，导致NIM跌破1.75%的同时信用成本上行50bp+。净利润由增转降，市场进一步下调估值至1.0x PB以下。目标价35 CNY（-10.8%，股息部分对冲后约-5%）。

**主要监控指标：**
- 季报净息差（每季披露）：若低于1.75%，Adverse概率上升
- 财富管理手续费增速（每半年报）：若低于0% YoY，护城河逻辑动摇
- 不良贷款率（每季）：若超过1.2%（vs当前~0.9%），信用成本优势受损
- 2025全年分红公告：若每股分红低于1.8 CNY（预期约2.03 CNY），政策收紧信号

## Valuation — Risk/Return 评估

> 回答：当前定价是否合理？承担了多少风险，预期换取多少收益？这是量化的维度，不是预测，是纪律的来源。

**估值框架：**
银行股核心锚：P/B + P/E双重验证。招行作为质量溢价银行，历史合理P/B区间1.3-2.0x，历史P/E区间7-12x。当前P/B约1.1x、P/E 6.8x，均处于历史底部区间（参照2022年底、2024年初低点水平）。

**核心数据（来源：JQData/同花顺，21家分析师，2026年3月）：**
| 年度 | EPS（CNY） | Net Profit（亿） | Forward P/E |
|------|-----------|----------------|-------------|
| 2025E | 5.81 | 1,500.08 | 6.8x |
| 2026E | 6.00 | 1,545.82 | 6.5x |
| 2027E | 6.26 | 1,611.71 | 6.3x |

- 当前股价：39.25 CNY（≈ $5.70 USD，FX 0.1452）
- 估计2025年末每股净资产（BPS）：约35-36 CNY，P/B ≈ 1.09-1.12x
- 预计2025年全年每股分红：约2.03 CNY（EPS 5.81 × 35%），股息率约5.2%
  （已于2026年1月派发2025年中期A股分红1.013 CNY/股）

**当前定价反映的预期：**
市场以6.8x PE定价了"息差持续恶化 + 盈利增长近乎停滞"的悲观情景，几乎没有给财富管理护城河任何估值溢价，也未给资产质量领先的信用成本优势溢价。这是本Thesis的核心预期差：如果财富管理费收入能维持10-15%增速，即便NIM小幅收窄，净利润仍可实现中个位数增长，不支持6.8x这一"零增长定价"。

**Risk/Return 评估（18个月Horizon）：**
| 场景 | 概率 | 目标价（CNY） | 持有总回报（含~7.5%股息） |
|------|------|-------------|----------------------|
| Bear（NIM<1.75%，信用成本上行） | 25% | 35 CNY | ≈ -5% |
| Base（NIM企稳，财富管理复苏） | 55% | 48 CNY | ≈ +28% |
| Bull（资本市场强劲复苏，AUM超增） | 20% | 60 CNY | ≈ +58% |
| **概率加权** | — | **≈49 CNY** | **≈ +26%** |

- Upside（Base兑现）：+28%
- Downside（Bear兑现）：-5%（股息覆盖大部分资本损失）
- 胜率：75%（Base+Bull），不对称明显偏多

**估值锚点：**
- 底部支撑：1.0x PB ≈ 35-36 CNY（历史底部）
- 合理中枢：1.3x PB ≈ 46-47 CNY（历史均值下限）
- 充分定价：1.6x PB ≈ 57-58 CNY（需要资本市场明显催化）

## Technical — 资金流与市场结构

> 回答：谁在交易这支股票？反映了什么预期？当前定价是怎么形成的？这是独立于 Fundamental 和 Valuation 的维度，时效性强。

**市场参与者结构：**
招行是A股银行板块核心权重股，机构（公募基金、险资、外资QFII/沪港通）主导定价，散户比例相对银行板块均值更低。险资是近年最重要增量：在利率持续下行的背景下，招行5%+股息率对负债端需匹配长久期资产的险资极具吸引力，构成持续性买盘。外资QFII/沪港通持仓对全球风险偏好敏感，是短期波动的边际定价力量。

**当前主要叙事 / 市场定价逻辑：**
市场主叙事：净息差收窄尚未见底 → 利息净收入增长停滞 → 给予历史低估值。2025年3月27日年报披露后股价当日大跌-5%，反映机构对息差指引的失望情绪仍未消化。市场尚未充分定价"大财富管理护城河"（AUM管理费的结构性增长）和"信用成本低波动优势"这两个中长期差异化因子。

**2026年YTD价格结构（JQData，截至2026-03-24）：**
- 区间：37.31 - 40.36 CNY；当前39.25 CNY（+4.67% YTD）
- 均量：约9,494万股/日，流动性充足
- 2026年3月17日公告：将于3月27日召开董事会审议2025全年业绩及年度股息，是近期最重要催化剂
- 4月15日：拟赎回2.75亿股优先股（共275亿元面值），减少优先股股息支出，对普通股东中性偏正

**潜在资金流变化：**
- **正面**：年报+年度分红宣告（3月27日）若符合或超预期，险资/高股息策略资金流入
- **正面**：国家队（宽基ETF/社保）在银行板块维稳期间持续配置
- **负面**：若2026年再现超预期降息（LPR下调30bp+），NIM担忧重燃，机构减仓
- **负面**：市场风格切换至科技/AI题材，高股息红利板块资金分流

**与Fundamental / Valuation的一致性：**
三维度当前格局：基本面中性偏正（财富管理高增 vs NIM压力），估值极低（6.8x PE/1.1x PB），市场情绪偏悲观（息差叙事主导）。这是一个有利的不对称设置，最大不一致点在于：财富管理费+18.76%的高增速与市场赋予的"零增长定价"之间的预期差仍未弥合。

## 错误指标（Invalidation Conditions）

> 这是核心原则。哪些情况发生时，必须承认自己错了——没有台阶可下，没有时间找借口，必须立刻重新评估或止损。

- **若2026年任意两个连续季度NIM低于1.75%** → 息差压力超出预期，Base Scenario不成立，减仓至1%权重并重新评估
- **若任何一期季报不良贷款率超过1.2%（vs当前~0.9%）** → 资产质量护城河受损，"信用成本低波动"逻辑部分失效，降仓至1%以内
- **若财富管理手续费收入连续两个半年度出现负增长（YoY < 0）** → 大财富管理核心逻辑失效，Thesis最重要支撑不再成立，退出全部头寸
- **若2025全年每股分红低于1.8 CNY（vs预期约2.03 CNY）** → 分红政策显著收紧信号，重新评估股东回报与资本消耗平衡，考虑减仓
- **若持有至2027年9月（Horizon结束）股价仍低于38 CNY且无重大基本面改善** → 预期兑现时间严重偏离，重新审查，考虑释放仓位部署在更高胜率机会

## 头寸管理原则

> 在开仓前确立，不在市场波动中临时决定。

| 参数 | 设定 |
|------|------|
| 目标权重 | 3.5%（约 $3,444 USD / CNY 23,730，需加仓约200股） |
| 当前权重 | 1.74%（300股，CNY 11,775 / $1,710，低于目标，头寸未建满） |
| 加仓条件 | ①年报（3月27日）确认息差企稳且分红≥2.0 CNY，加至3.5%目标权重；②股价回调至37 CNY以下（≤1.05x PB），分批加仓至目标权重 |
| 减仓条件 | ①任一IC触发（见上）；②股价涨至55 CNY以上（约1.55x PB），估值充分定价，减至1%以内 |
| 退出条件 | 财富管理费收入连续两期负增长（核心IC触发）；或总组合有更高胜率机会且本标的估值优势不再明显 |
| Time Horizon | 18个月（至2027年9月） |
| 调仓说明要求 | 每次调仓必须更新此文档，写明触发原因及IC状态 |
