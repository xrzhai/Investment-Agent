# 紫金矿业（601899.SH）估值底稿
**日期：** 2026-03-25
**分析类型：** FY2026E 自下而上运营模型 + Forward PE 估值
**当前股价：** ¥32.20

---

## 一、核心结论

1. **原稿最关键的问题不是“偏保守”，而是口径混用。**
   FY2025 各分部利润贡献使用的是**税后净利润**口径，但原稿成本桥接又直接拿它去反推“运营成本”，导致 FY2025/2026 成本表述不一致，也把 `¥62.4` 的高值目标价误写成了机构均值。

2. **按更一致的 pretax 单位成本桥重算后，FY2026E 的中枢在 EPS `~¥3.0`，而不是默认拿 `¥2.79–¥3.29` 当 Base band。**
   在当前 thesis 所定义的 base regime 下，若全年大致维持现货锚 `金价 $4,427/oz`、`铜价 $11,750/t`、`USD/CNY 6.90`，FY2026E 归母 EPS 约 **`¥2.99`**。这高于 THS 一致预期 `¥2.59`，但只落在外部高位盈利区间 `¥2.8–3.0` 的上沿，并不离谱。

3. **当前价对应 FY2026E Forward PE 约 `10.8x`，仍有吸引力，但已不是 v3/v4 所写的“明显错误定价”。**
   更合理的判断是：**紫金仍有折价，但折价主要来自“全球矿业 forward 口径”与“A 股重估口径”之间的差**，而不是一个显著算错的超低估值。

4. **赤峰黄金不应再按“小额财务投资”处理，而应升级为“控制权收购待并表”模块。**
   按 2026-03-22 公告口径，紫金黄金将通过 A 股协议转让 + H 股定增认购合计取得赤峰黄金 **`25.85%`** 的股份并获得控制权。对估值的正确处理方式是：**紫金主业 EPS 单独保留，赤峰黄金并表影响单独做 pro forma 增量桥接。**

### 2.2 FY2025 分部桥接口径

为了避免税后利润与 pretax 成本混用，本次改用：

```text
FY2025 implied pretax profit = 分部税后净利润 / (1 - 25%)
FY2025 implied pretax cost   = 分部收入 - implied pretax profit

FY2026E pretax cost
= FY2025 implied pretax unit cost × FY2026E 产量
```

### 2.3 FY2025 隐含分部 pretax 成本

| 分部 | FY2025 收入 | FY2025 税后净利润 | 隐含 pretax 利润 | 隐含 pretax 成本 |
|------|-------------|------------------|------------------|------------------|
| 黄金 | `¥696亿` | `¥268亿` | `¥357.3亿` | `¥338.7亿` |
| 铜 | `¥761亿` | `¥255亿` | `¥340.0亿` | `¥421.0亿` |
| 其他 | `¥2,034亿` | `¥115亿` | `¥153.3亿` | `¥1,880.7亿` |

> 这一版的关键修正是：**成本桥接统一用 pretax 口径**。原稿里 `¥428 / ¥506 / ¥1,881` 与 `¥396 / ¥463 / ¥1,975` 同表并列，经济含义不一致，容易把税后利润差额误读成运营成本。

| 铜 | `¥421.0亿` | `109万吨 -> 120万吨` | `¥463.5亿` |

- “其他”业务利润不快于收入增长：`115 -> 120.8`，增速约 `5%`

### 3.3 固定成本摊薄敏感性

Base case **不预埋**固定成本摊薄收益。若仅对金、铜两项隐含 unit cost 给予 `-3%` 的摊薄敏感性，其余不变：

| 项目 | Base | Base + 3% 摊薄 |
|------|------|----------------|
| FY2026E EPS | `¥2.99` | `¥3.05` |
| 增量 | — | `+¥0.06` |

**含义：** 产量放大带来的 unit cost 摊薄确实存在，但量级不大，不能把它当成推升估值中枢的核心理由。

### 4.3 上沿盈利校验

外部盈利聚合页面给出的 FY2026E 高位盈利区间大致在 **`EPS ¥2.8–3.0`** 一带。

**如何理解本模型与卖方的差异：**

1. **本模型 Base = `¥2.99`，高于 THS `¥2.59`，但只高到外部上沿。**
2. 差异主要来自：**本模型把当前现货锚 (`$4,427 / $11,750`) 视作全年近似平均价**，而卖方均值通常更接近全年均价，不直接用 3 月现货价穿全年。
3. 因此，`¥2.99` 可以作为**“现货维持全年”的内部上中枢**，但不能把 THS `¥2.59` 简单定性为“明显错误”。

### 4.4 赤峰黄金控制权收购的估值处理

**公告口径（按 2026-03-22 交易方案）：**
- 紫金黄金受让 `241,925,746` 股 A 股，交易总价 `100.06 亿元`
- 紫金黄金按 `30.19 港元/股` 认购 `310,902,731` 股 H 股，认购金额约 `93.86 亿港元`
- 紫金体系交易前已持有 `18,833,400` 股
- 交易完成后合计持有 `571,661,877` 股，约占增发后总股本 **`25.85%`**

**算术检查：**
- A 股协议转让单价约 **`¥41.36/股`**
- H 股认购金额约 **`93.86 亿港元`**
- 增发后总股本隐含约 `22.11 亿股`
- 交易对价合计约 **`182.58 亿元人民币等值`**

**如何纳入模型：**

1. **先分开“主业”和“并购增量”。**
   紫金现有金铜主业 FY2026E Base EPS 仍以 **`¥2.99`** 为底，赤峰黄金新增控制权不直接改写这部分运营桥。

2. **新增一条并表增量桥。**
   若按 THS 对赤峰黄金的 `2026E 归母净利 47.42 亿元` 测算，紫金按 `25.85%` 穿透可归属利润约 **`12.26 亿元`**，折合紫金 EPS 约 **`+¥0.046`**。

3. **但不能忽略资金成本。**
   对价约 `182.58 亿元`，若按 `3%–4%` 年化资金成本粗算，年度资金占用约 `5.5–7.3 亿元`，对应 EPS 拖累约 **`-¥0.02 至 -¥0.03`**。

4. **净增厚应看作“几分钱级别”，不是“几毛钱级别”。**
   因此在 full-year close 且经营平稳的理想情形下，赤峰黄金并表对紫金 FY2026E 的净 EPS 增厚大致在 **`+¥0.02–0.03`** 区间；若 2026 年并非全年并表，实际增厚还会更低。

5. **结论上属于战略强化，不属于短期 EPS 重估。**
   这笔交易更重要的意义是：紫金获得了一个新的黄金平台控制权，提升了集团的黄金资产厚度和未来整合弹性；但在 12 个月估值层面，它首先是一笔**控制权并购**，而不是凭空增加的自由资产。

## 五、Forward PE 估值

### 5.1 估值口径

本次统一采用 **FY2026E Forward PE** 做主锚，不再混用“海外 forward + A 股 trailing”直接得出结论。

**可参考的 forward 档位：**

| 档位 | 含义 |
|------|------|
| `10x` | 全球矿业去风险情景 |
| `13x` | 全球大型金矿合理中枢 |
| `15x` | 国内大型资源股正常 forward 估值 |
| `18x` | A 股资源重估 / 高景气 re-rating |

### 5.2 目标价矩阵（¥/股）

|  | **Bear EPS ¥2.07** | **Base EPS ¥2.99** | **Bull EPS ¥3.74** | **P 加权 EPS ¥2.99** |
|---|---:|---:|---:|---:|
| `10x` | `¥20.7` | `¥29.9` | `¥37.4` | `¥29.9` |
| `13x` | `¥26.9` | `¥38.9` | `¥48.6` | `¥38.9` |
| `15x` | `¥31.1` | `¥44.9` | `¥56.1` | `¥44.9` |
| `18x` | `¥37.3` | `¥53.8` | `¥67.3` | `¥53.8` |

### 5.3 关键目标价锚

| 锚点 | 计算 | 目标价 | 较现价 |
|------|------|--------|--------|
| 保守 | `13x × ¥2.99` | `¥38.9` | `+21%` |
| 基准 | `15x × ¥2.99` | `¥44.9` | `+39%` |
| 乐观 | `18x × ¥2.99` | `¥53.8` | `+67%` |
| 卖方均值 | 外部 12m 平均目标价 | `¥49.27` | `+53%` |

> 卖方均值 `¥49.27` 对应本模型 FY2026E EPS `¥2.99` 的约 **`16.5x` forward PE**。这不是全球矿业 peers 的保守中枢，而是**温和 A 股 re-rating** 的口径。

### 5.4 并购后的 pro forma 直觉

若把赤峰黄金 full-year 并表的净增厚按 `+¥0.02–0.03 EPS` 纳入，则：

- Base pro forma EPS 大致从 `¥2.99` 上修到 **`¥3.01–3.02`**
- 对应 `15x` 基准目标价大致从 `¥44.9` 上修到 **`¥45.2–45.3`**

**因此：**
- 方向上应视作正面
- 量级上不足以改写 `¥22 / ¥45 / ¥67` 的整数目标价
- 真正值得跟踪的是后续整合、资金来源、赤峰黄金产量兑现和紫金是否进一步增持

## 六、结论

1. **逻辑上，紫金的估值仍成立，但不该再用“明显错杀”来描述。**
   当前价对应 FY2026E forward PE `10.8x`，低于内部 Base `15x` 锚，也低于外部目标价均值隐含的 `16.5x`。

2. **最合理的内部表述是：**
   - 若按全球矿业 forward 中枢给价，合理值约 `¥39`
   - 若按国内大型资源股正常 forward 给价，合理值约 `¥45`
   - 若 A 股出现明显资源重估，才有望走向 `¥54+`

3. **与 v4 相比，最大的修正不是“看空了”，而是把模型的会计口径拉直。**
   这使得 EPS 中枢从“靠经验插值和口径混用撑起来”变成了“靠产量、单价、pretax 成本桥和一致预期交叉验证撑起来”。

4. **赤峰黄金现在应被视作“并购控制权模块”，但短期仍不改写主目标价。**
   原因不是它不重要，而是：以当前交易对价看，这更像一笔会在未来几年兑现战略协同的控制权收购，短期对 FY2026 EPS 的净增厚只有几分钱级别。

## 七、模型局限

| 局限 | 影响 |
|------|------|
| FY2025 金/铜/其他分部利润仍为近似拆分，不是公司正式分部披露 | 对绝对 EPS 有一定估算误差 |
| Base 情景把 3 月现货价近似当全年均价 | 对 FY2026E EPS 偏乐观 |
| 其他业务按净利率持平处理，未单列银/锌/钼的弹性 | 低估或高估都可能 |
| 赤峰黄金并购尚处交易/审批/交割阶段，2026 年并表起点与融资结构未最终落地 | 对 FY2026 净增厚存在时点误差 |
| H 股 IPO / 再融资稀释未建模 | 若推进，EPS 会被摊薄 |
| 税率统一按 25% 桥接 | 与真实项目税率并不完全一致 |
